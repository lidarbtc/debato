## Usage on Debian

```sh
 sudo apt install cargo

 git clone https://github.com/lidarbtc/debato.git
 cd debato

 cargo build --release

 # 반드시 MYSQL 설정, rocket.toml 설정 후 실행

 ./target/release/debato
```
